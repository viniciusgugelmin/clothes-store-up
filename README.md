## Requirements
It is necessary to create a "db.properties" file outside the project folder.

### db.properties
```
user={{mysql_user}}
password={{mysql_password}}
dburl=jdbc:mysql://localhost:3306/{{mysql_db}}
useSSL =no
allowPublicKeyRetrieval=true
```
